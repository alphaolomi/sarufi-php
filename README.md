## Installation

You can install the package via composer:

```bash
composer require alphaolomi/sarufi-php
```

## Usage

```php
$sarufi = new Alphaolomi\Sarufi([]);

$sarufi->login('Hello, Alphaolomi!');


$sarufi->createBot('mybot!');
```
